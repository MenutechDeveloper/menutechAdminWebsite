PASOS TUTORIAL

1
Preparar Archivos

Es obligatorio que tu repositorio en GitHub tenga la estructura correcta para el despliegue. Debes crear un archivo README.md en la raíz y una carpeta llamada docs que contendrá todo el sitio web.

Dentro de la carpeta docs coloca todos tus archivos HTML, CSS, JS e imágenes.

2
Configurar GitHub Pages

Ve a Settings > Pages. En "Build and deployment", selecciona la rama main y asegúrate de elegir la carpeta /docs. Haz clic en Save.

En "Custom domain", ingresa tu dominio (ej: midominio.com) sin "www". Tras guardar, espera unos minutos y podrás activar el HTTPS (candadito).

3
Gestión DNS en GoDaddy

Entra a GoDaddy, busca el dominio del cliente y entra en la sección de DNS. Busca la opción de Registros DNS o Administrar DNS.

Haz clic en el botón Añadir un registro nuevo para comenzar la vinculación con GitHub.
