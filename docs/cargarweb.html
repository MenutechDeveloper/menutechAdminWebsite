Cargar Website

Sigue estos pasos para desplegar tu sitio en GitHub y conectar tu dominio.

1
Preparar Archivos

Asegúrate de que tu repositorio en GitHub tenga la estructura correcta. Es obligatorio crear un archivo README.md en la raíz y una carpeta llamada docs.

Dentro de la carpeta docs debes colocar todos tus archivos (HTML, CSS, JS, imágenes, etc.).

2
Configurar GitHub Pages

Ve a Settings > Pages. En la sección "Build and deployment", selecciona la rama main y la carpeta /docs, luego dale click en Save.

Más abajo, en "Custom domain", escribe tu dominio (ej: midominio.com) sin "www" y dale Save. Espera unos minutos para que el sitio se despliegue y puedas activar el certificado HTTPS (candadito).

3
Gestión DNS en GoDaddy

Inicia sesión en GoDaddy, ve a tu lista de dominios y busca el dominio del cliente. Haz clic en DNS y luego en la pestaña Registros DNS.

Haz clic en el botón Añadir un registro nuevo para comenzar a ingresar los datos de GitHub.

4
Ingresar Registros A

Debes crear 4 registros de tipo A. Para todos usa el nombre @ y un TTL de 600 segundos. Haz clic en cada dirección IP para copiarla rápidamente:
